Discussion 1

The Internet of Things (IoT) is a network of devices that produce/measure data for collection and that is shared with each other through connectivity with the internet (IBM, 2024). This can lead to data driven decision making that help homes and businesses make better informed decisions.

IoT devices create massive amounts of data and for this data to be useful the integrity and quality of the data needs to be ensured. Various factors can contribute to quality issues of the data, these being:

  • Data format variety
  • Inconsistent timestamps
  • Faulty devices
  • Defective data transmission
These factors leading to missing data, data duplication, data anomalies and data violation (Ding et al., 2022). The occurrence of these quality issues can lead to Type 1 and Type 2 errors in the analysis of the data if not detected or dealt with. To prevent errors occurring in the analysis, data cleaning can be done, this helps detect errors and repair them via different methods. It is important that any data cleaning that is done is well documented, and processes discussed for peer review and study replication in the academic setting (Huxley et al, 2020).
